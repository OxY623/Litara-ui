# OraNote

**OraNote** — это современное React-приложение на Vite с поддержкой Apollo, MUI и TailwindCSS, реализующее базовый функционал CRUD для заметок. Приложение позволяет авторизованным пользователям:

- Создавать новые заметки
- Просматривать список существующих заметок
- Редактировать содержание заметок
- Отмечать заметки как «Избранное»
- Удалять заметки

---

## Стек технологий

- **Frontend:** React 18, Vite
- **UI:** Material UI (MUI), TailwindCSS
- **GraphQL:** Apollo Client
- **State Management:** Apollo Cache + React Hooks
- **Сборка и разработка:** Vite с HMR
